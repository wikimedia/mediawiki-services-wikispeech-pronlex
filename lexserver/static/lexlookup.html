Lexserver

Look up words:

words=hund,häst

/lexicon/lookup?lexicons=wikispeech_lexserver_testdb:sv&words=hund,häst

Look up word using SQL 'LIKE':

wordlike=häst__

/lexicon/lookup?lexicons=wikispeech_lexserver_testdb:sv&wordlike=häst__

Look up words with validation issues

hasEntryValidation=true

/lexicon/lookup?lexicons=wikispeech_lexserver_testdb:sv&hasentryvalidation=true

Please note that it doesn't work to set to "false" and expect only entries without validation issues. hasEntryValidation=false simply doesn't mean or do anything.

transcriptionlike

Look up word dom where transcription includes string o:

/lexicon/lookup?lexicons=wikispeech_lexserver_testdb:sv&words=dom&transcriptionlike=%25o:%25&pp=yes

(For this to work properly, the Sqlite db must be instructed that PRAGMA case_sensitive_like=ON)

Look up lemmas

/lexicon/lookup?lemmas=kex&pp=y&lexicons=wikispeech_lexserver_testdb:sv
